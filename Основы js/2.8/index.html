Задание 4

let a = prompt("Первое число: ", 1);
let b = prompt("Второе число: ", 2);
alert(a + b);

Ошибка в том, что prompt возвращает введеные данные в виде строки, их необходимо преобразовать в числовой тип для правильного сложения 

            Исправленный код:
            let a = prompt("Первое число: ", 1);
            let b = prompt("Второе число: ", 2);
            alert(+a + +b); // Или alert(Number(a) + Number(b));
